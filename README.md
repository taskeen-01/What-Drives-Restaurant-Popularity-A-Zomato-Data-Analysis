### Project Overview

This project analyzes Zomato restaurant data to uncover insights into customer preferences, restaurant popularity, and online ordering behavior.
The goal is to simulate real-world business analysis and derive actionable insights for food delivery platforms.

### Objectives
- Identify the most popular restaurant types
- Analyze customer engagement using votes
- Understand rating distribution
- Compare online vs offline ordering behavior
- Discover patterns across restaurant categories

### Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

### Key Insights
- Most Common Restaurant Type
- Dining and Delivery dominate the dataset
- Highest Customer Engagement
- Certain categories receive significantly higher votes
- Ratings Distribution
- Most restaurants fall between 3.5 – 4.5 ratings
- Online Ordering Behavior
- Majority of restaurants support online ordering
- Mode vs Ratings
- Online ordering shows competitive or slightly better ratings
- Category vs Order Mode
- Some restaurant types still rely heavily on offline orders
  
### Business Impact
- Helps identify high-performing restaurant categories
- Supports decision-making for marketing strategies
- Highlights areas to improve online ordering adoption
